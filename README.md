
# Happy credit

It is an clone of https://www.klarna.com/us/deals/ page. 
## Tech Stack Used

 - Frontend
   - ReactJS
   - HTML5
   - CSS3
 - Backend
   - MongoDB
 - Deployement
   - Heroku 

## Instructions to follow in CLI 

 - Backend
   - npm install
   - npm start
 - Frontend
   - npm install
   - npm start
